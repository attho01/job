# 나의 행복한 직업 찾기!! 🌟

## 프로젝트 소개
직업상담을 통해 행복한 직업을 찾아주는 인터랙티브 웹페이지입니다.
사용자의 성격, 선호하는 활동, 일하는 방식에 대한 3가지 질문을 통해
맞춤형 직업을 추천해드립니다.

### 주요 특징
- ✨ 간단한 3개의 질문으로 빠른 적성 테스트
- 🎯 8가지 직업 조합 알고리즘
- 🎨 아름다운 그라데이션 디자인
- 📱 모바일 친화적 반응형 레이아웃
- 🎭 부드러운 애니메이션 효과
- 🔄 다시 테스트하기 기능

### 추천 직업 목록
이 웹페이지는 다음 8가지 직업을 추천합니다:
1. **광고 기획자** - 외향적, 창의적, 팀워크형
2. **프리랜서 디자이너** - 외향적, 창의적, 독립형
3. **프로젝트 매니저** - 외향적, 논리적, 팀워크형
4. **영업 컨설턴트** - 외향적, 논리적, 독립형
5. **게임 개발자** - 내향적, 창의적, 팀워크형
6. **작가 / 일러스트레이터** - 내향적, 창의적, 독립형
7. **소프트웨어 개발자** - 내향적, 논리적, 팀워크형
8. **데이터 분석가** - 내향적, 논리적, 독립형

## 사용 기술
- **HTML5**: 시맨틱 마크업 및 웹페이지 구조
- **CSS3**: 그라데이션, 애니메이션, 반응형 디자인
- **JavaScript (ES6)**: 이벤트 처리, DOM 조작, 동적 콘텐츠

## 파일 구조
```
프로젝트 루트/
├── index.html          # 메인 HTML 파일 (웹페이지 구조)
├── style.css           # CSS 스타일 파일 (디자인 및 레이아웃)
├── script.js           # JavaScript 파일 (기능 구현)
└── CLAUDE.md           # 프로젝트 문서 (이 파일)
```

## 실행 방법
1. 프로젝트 폴더에서 `index.html` 파일을 찾습니다
2. `index.html` 파일을 더블클릭하거나, 웹브라우저(Chrome, Firefox, Edge 등)로 파일을 엽니다
3. 웹페이지가 브라우저에서 실행됩니다

## 학습 포인트
이 프로젝트를 통해 다음을 배울 수 있습니다:

### HTML
- 시맨틱 태그 사용 (`<header>`, `<main>`, `<section>`, `<footer>`)

### JavaScript
- DOM 요소 선택 및 조작
- 이벤트 리스너 (click, keypress)
- 조건문을 활용한 로직 구현
- 객체를 이용한 데이터 관리
- 동적 HTML 생성 (innerHTML)

## 문제 해결

### 페이지가 제대로 표시되지 않아요
- 모든 파일(index.html, style.css, script.js)이 같은 폴더에 있는지 확인하세요
- 파일명과 확장자가 정확한지 확인하세요
- 브라우저 캐시를 삭제하고 새로고침(Ctrl+F5)해보세요

### 버튼을 눌러도 반응이 없어요
- 브라우저 개발자 도구(F12)를 열어 콘솔에 에러가 있는지 확인하세요
- 모든 질문에 답했는지 확인하세요 (라디오 버튼 선택)
- JavaScript 파일이 제대로 로드되었는지 확인하세요

### 스타일이 적용되지 않아요
- `index.html`에서 CSS 링크가 올바른지 확인하세요
- CSS 파일명이 `style.css`인지 확인하세요
- 파일 경로에 한글이나 특수문자가 있으면 문제가 될 수 있습니다

## 향후 개선 아이디어
프로젝트를 더 발전시킬 수 있는 아이디어들:

1. **더 많은 질문 추가** - 5~10개의 질문으로 확장하여 더 정확한 추천
2. **결과 공유 기능** - 결과를 이미지로 저장하거나 SNS 공유
3. **진행률 표시** - 질문 진행 상황을 보여주는 프로그레스 바
4. **애니메이션 개선** - 질문 전환 시 부드러운 페이드 효과
5. **다국어 지원** - 영어, 일본어 등 다른 언어 버전
6. **통계 기능** - 각 직업이 몇 번 추천되었는지 차트로 표시
7. **상세 정보** - 각 직업의 연봉, 전망, 필요 역량 등 추가 정보
8. **이메일 전송** - 결과를 이메일로 받아볼 수 있는 기능
9. **테마 선택** - 다크 모드, 라이트 모드 전환
10. **데이터베이스 연동** - 사용자 결과를 저장하고 통계 분석
