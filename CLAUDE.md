# 나의 행복한 직업 찾기!!

## 프로젝트 설명
직업상담을 통해 행복한 직업을 찾아주는 웹페이지입니다.
사용자의 관심사와 성향을 파악하여 적합한 직업을 추천해드립니다.

## 사용 기술
- **HTML**: 웹페이지 구조 및 콘텐츠
- **CSS**: 스타일링 및 레이아웃
- **JavaScript**: 동적 기능 및 사용자 상호작용

## 파일 구조
```
프로젝트 루트/
├── index.html          # 메인 HTML 파일 (웹페이지 구조)
├── style.css           # CSS 스타일 파일 (디자인 및 레이아웃)
├── script.js           # JavaScript 파일 (기능 구현)
└── CLAUDE.md           # 프로젝트 문서 (이 파일)
```

## 실행 방법
1. 프로젝트 폴더에서 `index.html` 파일을 찾습니다
2. `index.html` 파일을 더블클릭하거나, 웹브라우저(Chrome, Firefox, Edge 등)로 파일을 엽니다
3. 웹페이지가 브라우저에서 실행됩니다

## 코딩 규칙
이 프로젝트는 초보자를 위한 학습용 프로젝트입니다. 다음 규칙을 따라주세요:

### 1. 한국어 주석 사용
- 모든 주석은 한국어로 작성합니다
- 코드의 목적과 동작을 명확하게 설명합니다

### 2. 초보자가 이해하기 쉽게 코드 작성
- 복잡한 코드보다는 간단하고 명확한 코드를 작성합니다
- 변수명과 함수명은 의미를 쉽게 알 수 있도록 작성합니다
- 한 번에 하나의 기능만 구현합니다

### 3. 코드마다 설명 주석 달기
- 각 HTML 섹션에 설명 주석을 추가합니다
- CSS 속성 그룹마다 설명을 추가합니다
- JavaScript 함수마다 기능 설명을 추가합니다

### 주석 예시
```javascript
// 사용자가 버튼을 클릭했을 때 실행되는 함수
function handleButtonClick() {
    // 입력값을 가져옵니다
    const userInput = document.getElementById('input').value;

    // 결과를 화면에 표시합니다
    displayResult(userInput);
}
```
